HackerOS

HackerOS, built on Debian Testing, is a high-performance distro tailored for gamers, developers, cybersecurity experts, and everyday users.

Website: HackerOS Website

LegendaryOS

LegendaryOS, based on Debian, offers Arch Linux on Debian, meaning it has an installer modeled after the Arch-install installer. The installer is called legendary-installer. Additionally, LegendaryOS has a community repository called LCR, or legendary community repository.

Website: LegendaryOS Website

Nula Programming Language

The Nula programming language is a beta project. It has a .Nula file and a CLI tool, Nula, written in Go, Zig, and Rust. Nula is as efficient as Rust but much simpler. Additionally, you can write in another language using #. The Nula programming language has many interesting features.

Website: Nula Website

Universal Live Builder

Universal Live Builder is used to create your own distribution on any parent distribution. It allows you to create a classic system or an atomic system, or it has many options for creating your own distro.

Website: ULB Website
